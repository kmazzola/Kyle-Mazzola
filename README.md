# Flashcard Study Application

A Python program that helps me study for my Managerial Accounting exam using flashcards.

## Features
- Reads flashcards from a text file
- Randomizes card order
- Studies in batches of 10 cards
- Reviews wrong answers immediately after each batch
- Tracks overall performance

## How to Run
1. Make sure you have Python installed
2. Download both `flashcard_app.py` and `flashcards.txt`
3. Place both files in the same folder
4. Run: `python flashcard_app.py`

## Requirements
- Python 3.x
- No external libraries needed (uses built-in random module)
